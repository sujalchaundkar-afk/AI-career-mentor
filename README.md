 # AI Career Mentor

# About the Project

This project is developed for a hackathon under the **Sustainability & Social Impact** theme.

Many students are confused about choosing the right career because they do not have proper guidance. The aim of this project is to build a simple AI-based web application that helps students explore career options based on their interests and skills.

## Problem

Students often don't know:
- Which career suits them.
- What skills they need to learn.
- Where to start.

This can lead to poor career decisions and wasted time.

## Our Solution

AI Career Mentor allows users to enter their interests and skills. Based on this information, the application suggests suitable career paths and recommends skills that can help them improve.

This project is being developed as a learning project and can be improved by adding more AI features in the future.

## Features

- Career suggestions
- Skill recommendations
- Simple and easy-to-use interface
- Fast responses

## How to Run

1. Clone this repository.
2. Install Flask using:

```bash
pip install -r requirements.txt
```

3. Start the application:

```bash
python app.py
```

4. Open the link shown by Flask in your web browser.
